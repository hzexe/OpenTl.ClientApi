OpenTl.ClientApi

The Api client library that implements the MtProto 2.0 protocol is supported by .Net 4.6 and the dotnet core >= 1.6. Suitable for the realization of clients for work with servers that support the MtProto protocol, such as Telegram

It's a perfect fit for any developer who would like to send data directly to Telegram users or write own custom Telegram client.

We are used the 76th schema layer

Install via NuGet
